# Wayback On Cloudflare

**This project has suspended due to the Cloudflare Worker scripts upload limit.**

Host wayback service on Cloudflare using Worker.

## Development

```bash
# Install Node modules
yarn

# Build wayback WASM
make build

# Build JavaScript library
yarn run build

# Preview service
yarn run preview
```
